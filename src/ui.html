Welcome to Boolean

This is an AI agent that can run commands and edit files on this PC — always with your approval unless you enable auto-approve.

AI makes mistakes. Review what it asks to do. The software is provided as-is, without warranty; you are responsible for what runs on your machine.

Local mode runs on your PC. Cloud mode sends relevant content to the third-party provider whose API key you choose. Account sign-in is separate from AI access. Full terms: LICENSE.txt and PRIVACY.txt in the install folder.
