6. SYSTEM REQUIREMENTS AND INCLUDED RUNTIMES
Boolean is intended for Windows 10 or Windows 11 on x64 PCs. The installer
includes the Boolean app, embedded Node.js backend, local llama.cpp CPU
inference engine, and required app-local runtime DLLs. Users do not need to
install Node.js, .NET, Ollama, Python, or the Microsoft Visual C++
Redistributable to run the standard local version.

Boolean uses Microsoft Edge WebView2 for the app window and built-in browser.
Most Windows 11 PCs already include WebView2. If WebView2 is missing, Windows
or the installer/store may need to install the WebView2 Runtime before Boolean
can display its interface.

Minimum system requirements depend on how Boolean is used:
  (a) Cloud mode: 4 GB RAM, internet connection, WebView2, and a supported
      third-party provider API key.
  (b) Lightest local LLM mode: 8 GB RAM recommended, 2 GB free disk space, and
      a CPU with AVX2 preferred.
  (c) Starter model mode: very small bundled models such as SmolLM2-135M may
      run on lower-end PCs, but answer quality and tool reliability are limited.
  (d) Recommended local agent use: 16 GB RAM and 20 GB free disk space for
      models, project files, downloads, and update staging.
