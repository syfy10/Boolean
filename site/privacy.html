Privacy

Updated July 18, 2026

Boolean is built for local-first work. Standard local chats, notes, browser state, settings, model files, screenshots, and project files stay on your Windows PC when you use a local model.

If you choose cloud features, sign in to a Boolean account, add a provider key, connect a remote MCP server, browse the web, download a model, or ask an online model to inspect a screenshot, page, note, or project file, the information needed for that request is sent to the selected service. Cloud AI uses your chosen provider key and does not require Boolean sign-in. Each service's privacy policy, retention rules, terms, and charges apply.

Provider keys and MCP tokens are stored in your local Boolean configuration. Protect that file like a password. Boolean's state API reports only whether a secret exists, not the secret itself.

Boolean checks saz3 Labs GitHub releases for updates. Update checks can reveal normal connection information such as IP address, app version, and user-agent, but do not upload chats, files, API keys, notes, or models.

Boolean's public source and Windows releases are available at github.com/syfy10/Boolean.

For support, contact saz3labs@gmail.com.
